# 维护保养

保持设备状态良好,数据才稳。重点是**娇贵的视触觉传感器表面**与**标定有效性**。

!!! warning "以硬件团队说明为准"
    传感器清洁剂、耗材更换、返修流程等**具体规格**需硬件团队确认。本页先给通用安全做法与
    骨架,标「待补」处待补充。

## 视触觉传感器(最需要小心)

视触觉传感器的成像面是**弹性体/凝胶类材料,较娇贵**,直接决定触觉数据质量。

- **避免**:尖锐物刮擦、指甲抠压、硬物撞击、长时间重压变形、油污与溶剂。
- **清洁**:用干净的**无绒软布**轻拭;顽固污渍的可用清洁剂**待硬件团队确认**——
  在确认前不要自行使用酒精/溶剂。
- **存放**:避光、常温、**避免长期受压**导致表面变形;不用时加保护(镜头盖/软垫,待补)。
- **接触**:采集时正常接触物体即可,避免异常大力挤压。

!!! danger "不要做"
    不要用尖锐物清理表面、不要撕扯贴合面、不要在未确认的情况下用溶剂擦拭。

## 相机镜头

- 腕相机与触觉光学面用**气吹 + 无绒布**清洁,避免刮花。
- 保持镜头清洁,过脏会直接降低图像质量。

## 编码器与标定

- **定期重标零点**:确认闭合时 `gripper.pos`≈0;偏了就重标(见 [4.1 编码器零点标定](04-calibration.md#41))。
- **跌落/大力冲击后**必须重标,并检查机械是否松动。
- 建议在每个采集批次前做一次快速自检([一页速通 §2](quickstart.md))。

## 线缆与接口

- USB / 串口线**避免反复弯折**、避免拉扯根部;插拔轻柔。
- 固定走线,减少采集中被牵扯导致的接触不良/掉线。

## 通用

- 避免跌落、进液、极端温湿度。
- 长期不用**断电存放**,置于干燥避光处。

## 保养周期建议(待硬件团队给准)

| 项目 | 建议周期 | 备注 |
|---|---|---|
| 传感器表面检查/清洁 | *待补* | 目视有无划痕/污渍 |
| 编码器零点复标 | 每批次前 / 冲击后 | 闭合≈0 |
| 镜头清洁 | *待补* | — |
| 线缆检查 | *待补* | 有无破损/松动 |

## 待补(需硬件团队)

- 允许使用的清洁剂与方法
- 传感器表面/耗材更换与寿命
- 返修 / 送检流程与联系人
